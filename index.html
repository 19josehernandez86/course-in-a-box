What is this?

This is an experiment in mixing the creation of a course and the running of a course in a way that would be hard to distinguish using Github pages

Things you may learn includes creating a site using Jekyll and GitHub pages, Markdown, theories of social learning and most importantly, some other great people!

How do I sign up?

  1. Fork this repository on GitHub. You can do so by clicking here
  2. Next up you need to edit _data/course.yml. Add your name, image URL and Twitter handle.
  3. Submit a pull request to get your changes merged into the main version of the course and have your mug show up here! See this pull request for an example.
